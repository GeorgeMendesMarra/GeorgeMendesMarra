## Questões sobre NPU (Neural Processing Unit)

---

**Questão 1**
O que significa a sigla NPU?

a) Network Processing Unit

b) Neural Processing Unit

c) Numerical Processing Unit

d) Node Processing Unit

e) Native Programming Unit

**Resposta correta: b**

---

**Questão 2**
Qual é a principal função de uma NPU?

a) Executar sistemas operacionais

b) Processar gráficos 3D em alta resolução

c) Acelerar operações matemáticas de redes neurais

d) Gerenciar a memória RAM do sistema

e) Controlar dispositivos de entrada e saída

**Resposta correta: c**

---

**Questão 3**
Qual operação matemática é a mais fundamental e acelerada por NPUs?

a) Divisão de ponto flutuante

b) Raiz quadrada

c) Multiplicação-acumulação (MAC)

d) Logaritmo natural

e) Seno e cosseno

**Resposta correta: c**

---

**Questão 4**
Qual é a unidade básica de processamento dentro de uma NPU?

a) ALU (Arithmetic Logic Unit)

b) CUDA Core

c) MAC (Multiply-Accumulate)

d) FPU (Floating Point Unit)

e) AGU (Address Generation Unit)

**Resposta correta: c**

---

**Questão 5**
Qual arquitetura de processamento é comumente utilizada em NPUs para multiplicação de matrizes?

a) Von Neumann

b) Harvard

c) Systolic array (arranjo sistólico)

d) Superscalar

e) VLIW (Very Long Instruction Word)

**Resposta correta: c**

---

**Questão 6**
Qual formato de precisão é MAIS utilizado em NPUs para inferência de redes neurais?

a) FP64

b) FP32

c) INT8

d) FP128

e) INT32

**Resposta correta: c**

---

**Questão 7**
Qual a principal vantagem das NPUs sobre as GPUs para inferência local?

d) Maior frequência de clock

e) Capacidade de treinar redes neurais
