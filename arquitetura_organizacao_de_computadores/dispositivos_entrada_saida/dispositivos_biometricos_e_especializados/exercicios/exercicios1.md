## 🔍 Curiosidades Históricas

| Curiosidade | Detalhe |
| :--- | :--- |
| **O nome "Eigenfaces"** | Deriva de "eigenvalues" (autovalores) da álgebra linear, base matemática do método |
| **A primeira condenação** | Francisca Rojas, Argentina, 1892 — cortou a própria garganta para simular ataque após matar os filhos |
| **A íris de Daugman** | O algoritmo de Daugman é usado em sistemas de identificação em aeroportos e fronteiras de mais de 50 países |
| **Face ID e privacidade** | O Face ID processa dados no dispositivo (Secure Enclave), não na nuvem da Apple |
| **Biometria em passaportes** | Desde 2006, passaportes eletrônicos (e-passports) incluem dados biométricos faciais e de impressões digitais |
| **O olho do futuro** | A Índia possui o maior banco de dados biométrico do mundo (Aadhaar), com mais de 1,2 bilhão de registros |
