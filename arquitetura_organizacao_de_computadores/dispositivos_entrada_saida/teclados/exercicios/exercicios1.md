## 📚 Resumo dos Principais Conceitos Abordados

| Conceito | Descrição |
|----------|-----------|
| **QWERTY** | Layout desenvolvido por Sholes em 1868 para evitar travamento das máquinas de escrever |
| **IBM Model M (1985)** | Teclado com switches buckling spring; considerado o padrão ouro da indústria |
| **Switches Cherry MX** | Padrão da indústria para teclados mecânicos: Red (linear), Brown (tátil), Blue (clicky) |
| **Mecanismo tesoura** | Tecnologia predominante em notebooks; perfil baixo e estável |
| **ABNT2** | Layout brasileiro com tecla "ç" e Enter em "L" invertido |
| **N-key rollover (NKRO)** | Capacidade de registrar múltiplas teclas simultaneamente |
| **Hot-swappable** | Permite troca de switches sem solda |
| **QMK/VIA** | Firmware open-source para customização avançada |
| **Switch óptico** | Utiliza sensor de luz em vez de contato metálico |
| **PS/2** | Conector padrão para teclados no final dos anos 1980 e 1990 |
| **DEC VT100 (1978)** | Terminal que estabeleceu padrões de teclas de função e navegação |
