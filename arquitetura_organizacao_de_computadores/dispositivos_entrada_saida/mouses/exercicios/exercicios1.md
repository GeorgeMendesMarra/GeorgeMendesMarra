## 📚 Resumo dos Principais Conceitos Abordados

| Conceito | Descrição |
|----------|-----------|
| **Douglas Engelbart (1964)** | Inventor do primeiro protótipo de mouse no Stanford Research Institute |
| **The Mother of All Demos (1968)** | Demonstração histórica que apresentou mouse, hipertexto e videoconferência |
| **Xerox Alto (1973)** | Primeiro computador com mouse e interface gráfica (não comercial) |
| **Xerox Star 8010 (1981)** | Primeiro computador comercial com mouse |
| **Apple Macintosh (1984)** | Popularizou o mouse ao torná-lo acessível ao público |
| **Mouse mecânico (bola)** | Tecnologia predominante antes do óptico; usava bola de borracha e roletes |
| **Microsoft IntelliMouse Explorer (1999)** | Primeiro mouse óptico moderno sem necessidade de mouse pad |
| **Sensor óptico (LED)** | Ilumina superfície com LED; sensor CMOS captura imagens |
| **Sensor a laser** | Funciona em mais superfícies, incluindo vidro |
| **DPI (Dots Per Inch)** | Medida de sensibilidade do mouse |
| **Polling rate** | Taxa de transmissão de dados; 1000 Hz = 1ms de latência |
| **PS/2** | Conector padrão para mouses no final dos anos 1980 e 1990 |
| **Palm grip / Claw grip / Fingertip grip** | Diferentes formas de segurar o mouse |
| **Mouse vertical** | Design ergonômico que reduz torção do antebraço |
| **Trackball** | Dispositivo com bola estacionária movida com os dedos |
| **PixArt** | Fabricante líder de sensores para mouses gamers |

## 🖱️ Linha do Tempo Visual dos Principais Marcos

```
1964 ── Engelbart cria primeiro mouse (madeira, uma roda)
1968 ── "The Mother of All Demos" apresenta o mouse ao mundo
1973 ── Xerox Alto: primeiro computador com mouse e GUI
1981 ── Xerox Star: primeiro mouse comercial
1984 ── Apple Macintosh: popularização do mouse
1991 ── Logitech MouseMan: primeiro mouse ergonômico em larga escala
1996 ── Microsoft IntelliMouse: introdução da roda de rolagem
1999 ── Microsoft IntelliMouse Explorer: primeiro mouse óptico moderno
2004 ── Logitech MX1000: primeiro mouse a laser sem fio
2010s ── Era dos mouses gamers: alta DPI, polling rate elevado, ultraleves
```

Estas questões abrangem desde a invenção pioneira de Douglas Engelbart até as tecnologias contemporâneas de sensores de alta precisão, proporcionando uma visão abrangente da história e evolução dos mouses de computador.
