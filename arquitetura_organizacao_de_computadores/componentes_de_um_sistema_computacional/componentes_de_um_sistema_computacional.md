### 1. Hardware (A Camada Física)

É o conjunto de componentes eletrônicos e mecânicos. o hardware é onde as políticas de controle de acesso físico e integridade de dados começam.
 
**Processador (CPU):** O "cérebro" do sistema, responsável por executar cálculos aritméticos e lógicos.


* **Memória RAM:** Memória volátil de alta velocidade que armazena os dados e programas em execução. É um ponto crítico para a **Confidencialidade**, pois ataques podem tentar ler dados sensíveis diretamente da memória.


* **Dispositivos de Armazenamento (HD/SSD):** Onde a informação é guardada de forma permanente. Aqui entra a importância da criptografia para garantir a proteção de dados em repouso.


* **Placa-Mãe:** O barramento que interliga todos os componentes citados acima.

---

### 2. Software (A Camada Lógica)

O software instrui o hardware sobre o que fazer. No seu cronograma, o desenvolvimento do **Blog do Aluno no GitHub** representa a aplicação prática dessa camada.

* **Sistema Operacional (SO):** Gerencia o hardware e fornece a interface para o usuário. Exemplos incluem Windows, Linux e macOS. O SO é responsável pela **Integridade** do sistema, controlando permissões de usuários.


* **Firmware:** Software básico gravado no hardware (como a BIOS/UEFI), essencial para o início seguro (Secure Boot) do computador.
* 
**Aplicativos e Linguagens de Programação:** Ferramentas usadas para criar soluções, como as tecnologias de **Back-End** mencionadas no seu plano para proteger sistemas.
