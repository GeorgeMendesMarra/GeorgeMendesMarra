## Questões sobre GPU (Graphics Processing Unit)

---

**Questão 1**
O que significa a sigla GPU?

a) Graphics Performance Unit

b) General Processing Unit

c) Graphics Processing Unit

d) Graphical Program Utility

e) Grid Processing Unit

**Resposta correta: c**

---

**Questão 2**
Qual a principal diferença arquitetural entre uma CPU e uma GPU?

a) CPU tem milhares de núcleos simples; GPU tem poucos núcleos complexos

b) CPU tem poucos núcleos complexos; GPU tem milhares de núcleos simples

c) CPU e GPU têm a mesma arquitetura, apenas mudam as instruções

d) CPU opera apenas em inteiros; GPU opera apenas em ponto flutuante

e) CPU não tem cache; GPU tem cache L3

**Resposta correta: b**

---

e) Unidades de aceleração de compressão de vídeo
