A **Representação da Informação** e a **Aritmética Computacional** são conceitos fundamentais da **Ciência da Computação**, pois explicam **como os computadores armazenam, representam e manipulam dados internamente**.

---

# Representação da Informação

A **representação da informação** refere-se à forma como os dados são codificados para serem processados pelo computador. Como os computadores funcionam eletronicamente, eles utilizam o **Sistema Binário**, que utiliza apenas dois valores: **0 e 1**, chamados de **bits**.

Esses bits podem representar diferentes tipos de informação, como:

* **Números**
* **Caracteres**
* **Imagens**
* **Áudio**
* **Vídeo**

### Unidades de informação

Algumas unidades importantes utilizadas na computação são:

* **Bit** – menor unidade de informação (0 ou 1)
* **Byte** – conjunto de 8 bits
* **Kilobyte (KB)** – aproximadamente 1024 bytes
* **Megabyte (MB)** – aproximadamente 1024 KB
* **Gigabyte (GB)** – aproximadamente 1024 MB

### Representação de caracteres

Para representar letras e símbolos, utilizam-se tabelas de codificação como:

* **ASCII**
* **Unicode**

Esses padrões associam cada caractere a um número binário específico.

---

# Aritmética Computacional

A **aritmética computacional** estuda **como os computadores realizam operações matemáticas utilizando circuitos digitais**.

Os computadores executam operações utilizando números representados em binário, e as operações são feitas pela **Unidade Lógica e Aritmética**, que faz parte da **CPU**.

### Operações básicas

As principais operações realizadas são:

* **Adição**
* **Subtração**
* **Multiplicação**
* **Divisão**

Essas operações são executadas através de circuitos lógicos compostos por portas lógicas.

### Representação de números

Os números podem ser representados de diferentes formas no computador:

* **Inteiros sem sinal**
* **Inteiros com sinal**
* **Ponto fixo**
* **Ponto Flutuante**

O padrão mais utilizado para números reais é o **IEEE 754**, que define como números decimais são representados e manipulados.

---

# Importância desses conceitos

A representação da informação e a aritmética computacional são essenciais porque:

* permitem que os computadores **armazenem dados corretamente**;
* possibilitam **processamento eficiente de informações**;
* são a base para **programação, sistemas operacionais e arquitetura de computadores**.

Esses conceitos estão presentes em praticamente todas as áreas da computação, desde o funcionamento do hardware até o desenvolvimento de software.

---

✅ **Resumo:**
A **representação da informação** define como os dados são codificados em binário para serem armazenados e processados pelo computador, enquanto a **aritmética computacional** estuda como as operações matemáticas são realizadas internamente pelos sistemas computacionais.
