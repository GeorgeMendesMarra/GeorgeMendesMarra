## Dicas para Resolver Rapidamente

| Tipo de questão | Método |
|----------------|--------|
| **Binário → Decimal** | Multiplicar dígitos por potências de 2 e somar |
| **Decimal → Binário** | Divisões sucessivas por 2, ler restos de baixo para cima |
| **Binário → Hexa** | Agrupar de 4 em 4 bits (da direita para esquerda) |
| **Hexa → Decimal** | Multiplicar dígitos por potências de 16 |
| **Complemento de 2** | Inverter bits → somar 1 (para negativo) ou se MSB=1: inverter bits → somar 1 → negar resultado |
| **Soma binária** | 0+0=0; 0+1=1; 1+1=0 vai 1 |
| **Subtração binária** | Se 0-1, empresta 1 do próximo bit |
| **Multiplicação binária** | Deslocamento e soma parcial |
