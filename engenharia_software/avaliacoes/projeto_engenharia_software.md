Modelo de Projeto de Software
Disciplina: Fundamentos de Engenharia de Software (Nível Superior)

1. Introdução
Este documento apresenta o modelo de projeto de software para aplicação dos fundamentos da Engenharia de Software, envolvendo análise, especificação, modelagem, projeto e qualidade de sistemas de software.
2. Tema do Projeto
Sistema de Gerenciamento de Biblioteca Acadêmica
3. Objetivos
3.1 Objetivo Geral
Aplicar os conceitos fundamentais da Engenharia de Software no desenvolvimento de um projeto de sistema.
3.2 Objetivos Específicos
- Elicitar e documentar requisitos funcionais e não funcionais
- Aplicar técnicas de modelagem UML
- Definir arquitetura básica do sistema
- Planejar atividades de desenvolvimento
- Aplicar conceitos de qualidade de software
4. Escopo do Sistema
O sistema permitirá o cadastro de usuários e livros, controle de empréstimos e devoluções e geração de relatórios básicos.
5. Entregáveis
- Documento de visão
- Documento de requisitos
- Diagramas UML
- Documento de arquitetura
- Plano de testes
- Apresentação final
6. Avaliação
O projeto será avaliado considerando a organização, clareza, aplicação correta dos conceitos de Engenharia de Software e participação da equipe.
