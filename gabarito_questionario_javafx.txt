Aqui está o gabarito das questões acima:

**Questão 1**

A resposta correta é **(D)**. JavaFX é uma biblioteca de gráficos e mídia para Java, um framework para desenvolvimento de aplicações desktop e uma plataforma para desenvolvimento de aplicativos móveis.

**Questão 2**

A resposta correta é **(D)**. As principais classes da API JavaFX são `Application`, `Scene`, `Stage`, `Node`, `Layout`, `Control`, `Image`, `Text` e `Button`.

**Questão 3**

A resposta correta é **(D)**. Para criar um formulário JavaFX, você precisa estender a classe `Application`, criar um objeto da classe `Scene` e adicionar componentes à classe `Stage`.

**Questão 4**

A resposta correta é **(C)**. Para adicionar um evento a um componente JavaFX, você precisa usar o método `setOnAction()`.

**Questão 5**

A resposta correta é **(D)**. Para conectar um formulário JavaFX a um banco de dados, você precisa usar o driver JDBC, a API JDBC ou a biblioteca MySQL Connector/J.

**Questão 6**

A resposta correta é **(D)**. Para criar uma animação JavaFX, você pode usar as classes `Timeline`, `KeyFrame` e `Interpolator`.

**Questão 7**

A resposta correta é **(D)**. Para criar um gráfico JavaFX, você pode usar as classes `LineChart`, `BarChart` e `PieChart`.

**Questão 8**

A resposta correta é **(D)**. Para criar um jogo JavaFX, você pode usar as bibliotecas JavaFX Scene Builder, JavaFX 3D ou JavaFX WebGL.

**Questão 9**

A resposta correta é **(D)**. As principais vantagens de usar JavaFX são:

* É uma biblioteca leve e eficiente
* É uma biblioteca multiplataforma
* É uma biblioteca fácil de aprender

**Questão 10**

A resposta correta é **(D)**. As principais desvantagens de usar JavaFX são:

* Não é tão popular quanto outras bibliotecas de gráficos
* Não é tão flexível quanto outras bibliotecas de gráficos
